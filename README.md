## How to build 

One time preparation:
```
npm install
npm install -g gh-pages
```

**Local development** (localhost:8000):
```
gatsby develop
```

**Production build and deploy**:
```
gatsby build
gh-pages -d public
```

Production build and deploy in one command:
```
npm run deploy
```
